## 2. 素材说明

| 文件                      | 用途                          | 注意                           |
| ------------------------- | ----------------------------- | ------------------------------ |
| `translation-glossary.md` | 统一人名、乐队名、术语翻译    | 写作时使用规范译名，简称可用   |
| `plot.md`                 | Ave Mujica 剧情事实依据       | 客观陈述，写作时加入自己的解读 |
| `mygo-plot.md`            | Ave Mujica 成员前史与关系起源 | 两作独立，不简单套用           |
| `plot-theories*.md`       | 了解观众多元解读，激发思考    | 非官方设定，区分事实与假说     |
| `staff-talk.md`           | 创作意图、技术细节            | 作者意图 ≠ 作品意义唯一来源    |
| `_characters/*.md`        | 角色背景、关系、心理状态      | 避免「贴标签」式简化           |
| `_songs/*.md`             | 歌词日文原文 + 中文翻译       | 结合剧情语境分析，注意双关     |

## 3. 已发布文章概要

### preface.md（前言）

核心论点：Ave Mujica 的「尚未完成」姿态需要读者共创意义。

主要理论框架：布尔迪厄（区隔）、接受美学（姚斯 / 伊瑟尔）、拉康（三界 / 匮乏 / 镜像）、布洛赫（希望哲学 / 具体乌托邦）、布莱希特（间离）、巴特 / 福柯（作者已死 / 人之死）、鲍德里亚（拟像）、利奥塔（宏大叙事终结）。

跨作对照：「来自新世界」（叙事独裁）、「进击的巨人」（具体乌托邦）、「少女革命」（革命的未完成）、「Remember11」（永恒阐释空间）。

核心隐喻：冰山、镜像、舞台 = 梦境 / 避难所、人偶 = 反人文主义主体观、「尚未」。
